### Secure Home Remote Access Architecture
Secure remote access system for a home development workstation using:

- Tailscale private networking
- SSH key authentication
- Wake-on-LAN automation
- Gateway-based device control

This project documents the architecture and configuration required to securely access a workstation from anywhere without exposing services to the public internet.
